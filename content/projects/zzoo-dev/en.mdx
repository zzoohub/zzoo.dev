## Content architecture

Each project has three document layers: product brief, case study, and design doc. All frontmatter is validated through Zod schemas. Image paths are sanitized. Slugs are type-checked.

Content is managed as MDX files split by locale — 6 languages (EN, ES, PT-BR, ID, JA, KO) with automatic fallback when a translation is missing.

## Search visibility

The site targets three discovery channels:

- **SEO** — Static generation, semantic HTML, XML sitemap with hreflang alternates, canonical URLs. Lighthouse 95+.
- **AEO** — JSON-LD schemas (`Person`, `WebSite`, `BlogPosting`, `SoftwareApplication`, `BreadcrumbList`) for AI answer engines. RSS feed for content indexing.
- **GEO** — `llms.txt` and clean MDX content for generative AI engines.

## Open source

The entire codebase is [open source](https://github.com/zzoohub/zzoo.dev). Built with Next.js 16, React 19, TypeScript, Tailwind CSS v4, deployed to Cloudflare Workers.
