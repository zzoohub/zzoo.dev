## Arsitektur konten

Setiap proyek punya tiga lapis dokumen: product brief, studi kasus, dan design doc. Semua frontmatter divalidasi melalui schema Zod. Path gambar disanitasi. Slug di-type-check.

Konten dikelola sebagai file MDX yang dibagi per locale — 6 bahasa (EN, ES, PT-BR, ID, JA, KO) dengan fallback otomatis ketika terjemahan belum tersedia.

## Visibilitas pencarian

Situs ini menargetkan tiga kanal penemuan:

- **SEO** — Static generation, HTML semantik, XML sitemap dengan hreflang alternate, URL kanonikal. Lighthouse 95+.
- **AEO** — Schema JSON-LD (`Person`, `WebSite`, `BlogPosting`, `SoftwareApplication`, `BreadcrumbList`) untuk mesin jawaban AI. Feed RSS untuk pengindeksan konten.
- **GEO** — `llms.txt` dan konten MDX bersih untuk mesin AI generatif.

## Open source

Seluruh codebase bersifat [open source](https://github.com/zzoohub/zzoo.dev). Dibangun dengan Next.js 16, React 19, TypeScript, Tailwind CSS v4, di-deploy ke Cloudflare Workers.
