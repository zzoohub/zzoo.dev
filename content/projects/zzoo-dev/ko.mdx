## 보이지 않는 개발자 문제

멋진 것들을 만들지만 아무도 당신을 찾지 못합니다. 포트폴리오는 한 달에 방문자 열몇 명 — 전부 이미 URL을 알고 있는 사람들. 그 사이 잠재 고객과 채용 담당자들은 구글에서 검색하고, ChatGPT에 물어보고, Perplexity에서 당신 같은 개발자를 찾고 있습니다.

이 채널들에 최적화되지 않으면, 당신의 스킬을 찾는 사람들의 99%에게 보이지 않습니다.

## 세 가지 발견 채널에 맞춰 설계

zzoo.dev는 그냥 예쁜 포트폴리오가 아닙니다. 발견되도록 엔지니어링되었습니다:

**SEO** — 정적 생성 페이지, 시맨틱 HTML, hreflang이 포함된 XML 사이트맵, 캐노니컬 URL. 인덱싱의 기술적 기본기.

**AEO** — JSON-LD 스키마(`Person`, `WebSite`, `BlogPosting`, `SoftwareApplication`, `BreadcrumbList`)가 AI 답변 엔진에 전문성에 대한 구조화된 맥락을 제공.

**GEO** — `llms.txt`와 깔끔한 MDX 콘텐츠로 "앱을 만들어줄 개발자가 누구?" 질문에 생성형 AI가 정확하게 답할 수 있도록.

## 오픈소스, 포크 환영

전체 코드베이스가 오픈소스입니다. Next.js 16, React 19, TypeScript, Tailwind CSS v4로 구축되어 Cloudflare Workers에 배포됩니다. 클론하고, 커스텀하고, 자신만의 버전을 배포하세요.
