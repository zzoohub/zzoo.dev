**차단 해제 버튼이 없다.**

이게 제품의 전부다. 다른 사이트 차단기에는 전부 탈출구가 있다 — 비밀번호, 타이머, "5분만 더" 팝업. 가장 이성적이지 못한 순간에 이성적으로 판단하기를 기대한다. Permanent Site Blocker는 그 선택지 자체를 없앤다.

사이트를 차단하면, 다시 접근하는 유일한 방법은 확장 프로그램을 완전히 재설치하는 것이다. 전체 차단 목록이 날아가고 처음부터 다시 등록해야 한다. 그 귀찮음이 핵심 기능이다.

## 무딘 게 아니라 정밀하게

YouTube 전체가 시간 낭비는 아니다 — Shorts가 문제다. Reddit 전체가 나쁜 것도 아니다 — 첫 페이지 토끼굴만 문제다. 진짜 방해되는 것만 골라서 차단한다:

- **전체 도메인** — `twitter.com`으로 모든 것 차단 (`www.` 포함)
- **특정 경로** — `youtube.com/shorts`로 Shorts만 차단
- **정규식 패턴** — `youtube\.com.*[가-힣]`로 한국어 YouTube 콘텐츠 차단

## 데이터는 브라우저 밖으로 나가지 않는다

분석 도구 제로. 외부 서버 제로. npm 의존성 제로. 전체 코드는 ~300줄 바닐라 JavaScript — 10분이면 전부 읽을 수 있다. 차단 목록은 Chrome 자체 스토리지로 기기 간 동기화된다. 그 외에 어떤 것도 데이터를 건드리지 않는다.

1,100명 이상이 매일 사용 중이다. 평균 평점 5.0. 대부분의 리뷰가 해제 버튼이 없다는 점을 설치 이유로 꼽는다.
