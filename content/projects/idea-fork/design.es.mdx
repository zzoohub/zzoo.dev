## Arquitectura

Idea Fork usa una arquitectura de ruta dividida: la web app de Next.js lee directamente de Neon PostgreSQL para todas las consultas de usuario, mientras que el backend FastAPI se ejecuta solo como pipeline de datos diario.

**Web App (Next.js en Vercel)**
- App Router con SSR/ISR, arquitectura Feature-Sliced Design (FSD)
- Acceso dual de datos: `@neondatabase/serverless` para lecturas SQL directas, Server Actions para escrituras (ratings)
- next-intl para 6 locales (en, es, pt-BR, id, ja, ko)
- Animaciones GSAP, tema oscuro/claro

**API + Pipeline (FastAPI en GCP Cloud Run)**
- Arquitectura hexagonal: patron de puertos/adaptadores de dominio con Protocol ABCs
- Pipeline diario de 5 etapas: Fetch, Tag, Score, Cluster, Synthesize
- Advisory lock previene ejecuciones superpuestas del pipeline

**Base de Datos (Neon PostgreSQL)**
- 10 tablas: post, tag, post_tag, cluster, cluster_post, brief, brief_source, product, product_tag, rating
- Paginacion keyset basada en cursor (cursores JSON codificados en base64)
- Indices parciales en `deleted_at IS NULL` para soft deletes

## Stack Tecnico

| Capa | Tecnologia | Razon |
|------|-----------|-------|
| Web | Next.js 16 + React 19 | App Router SSR/ISR, streaming, Server Actions |
| Estilo | Tailwind CSS v4 | Utility-first, modo oscuro con temas oklch |
| Animacion | GSAP + @gsap/react | Animaciones de scroll y entrada de alto rendimiento |
| API | Python 3.12 + FastAPI | Ecosistema rico en ML/LLM (scikit-learn, google-genai) |
| Arquitectura | Hexagonal (puertos/adaptadores) | Multiples adaptadores externos intercambiables |
| LLM | Google Gemini (Flash Lite + Flash + Embeddings) | Etiquetado, clustering y sintesis de briefs a bajo costo |
| Clustering | HDBSCAN (scikit-learn) | Deteccion automatica de numero de clusters |
| Base de Datos | PostgreSQL 18 (Neon) | Precio scale-to-zero, driver serverless |
| IaC | Pulumi (TypeScript) | Gestion declarativa para GCP, Neon y Vercel |
| Analiticas | PostHog | Analiticas de producto con 9 eventos personalizados |
| Errores | Sentry | Monitoreo de errores full-stack |
