## Arsitektur

**Mobile (React Native / Expo)**
- Expo Router untuk navigasi berbasis file
- Zustand untuk state klien, TanStack Query untuk state server
- MMKV untuk persistensi lokal yang cepat
- Dwibahasa (EN/KO) via i18next

**API (Rust / Axum)**
- SQLx dengan compile-time checked query terhadap Neon PostgreSQL
- Autentikasi JWT + JWKS (Google & Apple OAuth)
- Upload presigned URL ke Cloudflare R2
- Spek OpenAPI dengan Swagger UI

**Infrastruktur (Pulumi)**
- GCP Cloud Run dengan autoscaling (0–3 instance)
- Neon managed PostgreSQL
- Cloudflare R2 untuk penyimpanan foto
- GCP Secret Manager untuk kredensial
- GitHub Actions CI/CD untuk build dan deploy otomatis

## Deep Dive Tech Stack

| Layer | Teknologi | Kenapa |
|-------|-----------|--------|
| Mobile | React Native + Expo 55 | Cross-platform dengan performa native, update OTA via EAS |
| State | Zustand + TanStack Query | Pemisahan bersih state klien vs. server |
| API | Rust + Axum 0.8 | Type safety, footprint memori rendah, cold start cepat di Cloud Run |
| Database | PostgreSQL 18 (Neon) | Serverless Postgres dengan branching untuk dev/staging |
| Storage | Cloudflare R2 | Kompatibel S3, nol biaya egress untuk penyimpanan foto |
| IaC | Pulumi (TypeScript) | Manajemen infra deklaratif untuk GCP, Cloudflare, dan Neon |
| Monitoring | Sentry + PostHog | Error tracking + product analytics |
