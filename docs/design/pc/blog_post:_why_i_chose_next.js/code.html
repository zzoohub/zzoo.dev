Why I Chose Next.js for My Solopreneur Journey

React Engineering Architecture

When starting zzoo.dev, I faced the classic developer paralysis: which tech stack should I choose? The options are endless—Remix, Astro, plain React, or even simpler static site generators. However, as a solopreneur optimizing for velocity and performance, one framework stood out clearly above the noise.

The Need for Velocity

Time is the most expensive resource when you are a team of one. I needed a framework that provided sensible defaults without locking me into rigid patterns. create-next-app became my starting line, eliminating days of webpack configuration.

The hybrid rendering model—allowing me to mix Static Site Generation (SSG) for my blog posts and Server-Side Rendering (SSR) for my dynamic dashboard—was the killer feature. It meant I didn't have to compromise on SEO for content or interactivity for apps.

"The developer experience is unmatched when you need to ship fast. It feels less like a framework and more like a production-grade platform."

Configuring for Performance

Out of the box, the image optimization is incredible. But what really impressed me was how easy it was to customize the build process. Here is a snippet of my configuration to handle advanced caching headers and image domains.

next.config.js 
const nextConfig = {
  reactStrictMode: true,
  images: {
    domains: ['cdn.zzoo.dev', 'avatars.githubusercontent.com'],
    formats: ['image/avif', 'image/webp'],
  },
  // Experimental features for faster builds
  experimental: {
    serverActions: true,
  },
};

module.exports = nextConfig;

The Ecosystem Advantage

Beyond the code, the ecosystem is vast. Whether it's Vercel for hosting, simplified authentication with Auth.js, or styling with Tailwind CSS (which this site uses heavily), everything just works. The friction between idea and deployment is nearly zero.

Ultimately, choosing Next.js wasn't just a technical decision; it was a business decision. It allows me to spend less time configuring tools and more time building value.
